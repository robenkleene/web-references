# YouTube

## Playback

- `␣` / `k`: Play pause
- `l` / `j`: Forward / backward 10 seconds
- `←` / `→`: Forward / backward five seconds
- `,` / `.`: Forward / backward one frame

## Other

- `C`: Toggle captions

## Notes

- Example URL with time stamp: `https://www.youtube.com/watch?v=XAKQvcqxFbo&t=25m35s`
